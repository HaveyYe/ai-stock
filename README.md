# 股票量化策略使用说明

## 项目结构

```
AiStock/
├── stock_selector.py      # 选股脚本
├── download_stock_data.py # 数据下载脚本
├── backtest.py            # 回测系统
├── strategy_select_stocks.py  # 同花顺策略文件
└── stock_data/            # 股票数据目录（运行后生成）
```

## 使用步骤

### 1. 下载股票数据

```bash
python download_stock_data.py
```

这会下载2025年1月1日至今的所有主板股票数据到 `stock_data/` 目录。

### 2. 运行选股

```bash
python stock_selector.py
```

每天运行，选出符合条件的5只股票（昨日开盘30分钟内涨停+主板非ST非科创建非创业+集合竞价金额/市值排序）。

### 3. 运行回测

```bash
python backtest.py
```

使用已下载的数据进行回测，计算策略收益。

## 回测策略说明

### 选股条件
1. 昨日开盘30分钟内（9:25-9:55）首次涨停
2. 主板股票（非ST、非科创板68开头、非创业板30开头、非北交所4/8开头）
3. 按昨日成交额/总市值从大到小排序
4. 选取前5只股票

### 交易逻辑
- 开盘买入：用20%仓位买入每只股票（5只股票各20%，共100%）
- 止盈：收益率达到10%时卖出
- 止损：亏损达到10%时卖出
- 每天重新选股，每天交易

### 回测参数
- 初始资金：100000元
- 回测区间：2025-01-01 至今
- 交易成本：万分之二手续费
